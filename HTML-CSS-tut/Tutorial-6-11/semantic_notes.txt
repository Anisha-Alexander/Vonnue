SEMANTIC TAGS 
--------------


<main> 
For the main content of a webpage, unique to that page 

<section>
Defines a certain section of a webpage (eg. bloglist, contact info)

<article>
Defines  a bit of content which makes up an article (eg. blog post) 

<aside>
Defines some content related to something else (eg. similar blogs)

<footer>
For the footer of a website
